# yeek-vue

## 介绍
参考 PanJiaChen 大佬的 [vue-element-admin](https://github.com/PanJiaChen/vue-element-admin) 项目仿造而成的。

Yeek.top - vue 版，目前后端为 php ，有想做后端的可以联系我嗷嗷嗷。

### 使用
账号密码随意填写，唯有管理员账号 `admin` 有 作业-发布 的界面。

### 进度
目前 完成了前端的 `权限管理`、`登录`、`侧边栏渲染`、`基本作业表格`。
